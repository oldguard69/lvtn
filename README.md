# Graduated project

## Plagiarism detection system in Vietnamese.
## Technologies used:
- PhoBERT
- Flask
- Angular

## Instruction:
- Install pytorch with cuda:
    ```pip install torch -f https://download.pytorch.org/whl/torch_stable.html```
    
- In server:
    ```set FLASK_APP=server```
    ```flask run```

- In client:
    ```ng serve```
